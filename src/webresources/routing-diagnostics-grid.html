⚙ Settings

Auto-Update In-Progress Records
Enabled: Records that were previously retrieved and are still not completed will continue to be refreshed automatically, so you can see the latest status without clicking Retrieve again.

Disabled: Previously retrieved records will not be updated automatically. You will need to click Retrieve again to get the latest status.
Note: Automatic updates can only refresh records from the last 7 days.
Auto-Retrieve Diagnostics
Enabled: The system automatically retrieves and refreshes routing diagnostics from Application Insights on a recurring schedule.

Disabled: Data will only be retrieved when you manually click Retrieve.
Show Copilot
Enabled: The Routing Insight Copilot button is visible in the toolbar, allowing users to open the AI assistant panel.

Disabled: The Copilot button is hidden.
Open Copilot by Default
Enabled: The Copilot panel opens automatically when the page loads.

Disabled: The Copilot panel stays closed until manually opened.
Settings are stored as environment variables in your Dataverse solution.
